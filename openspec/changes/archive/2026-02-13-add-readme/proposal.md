## Why

專案即將發佈到 GitHub，目前缺少 README 文件。沒有 README 的 repo 無法讓訪客快速了解專案用途、安裝方式和技術架構。

## What Changes

- 新增 `README.md`（英文版），包含專案介紹、功能特色、快速開始、架構說明、開發指令
- 新增 `README.zh-TW.md`（繁體中文版），內容對應英文版，使用台灣用語
- 兩份 README 互相連結，讀者可切換語言
